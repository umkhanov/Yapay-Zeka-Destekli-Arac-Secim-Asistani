# Yapay-Zeka-Destekli-Arac-Secim-Asistani

Bu proje, Google Gemini API kullanarak kullanıcıya araba önerisi yapan bir web uygulamasıdır.

## Kullanılan Teknolojiler
- HTML / CSS / JavaScript (Frontend)
- Flask (Backend)
- Google Gemini API
- Render.com (Hosting)
- GitHub Pages (Frontend hosting)

## Çalışma Mantığı
1. Kullanıcı bir istek yazar.
2. Frontend isteği backend'e gönderir.
3. Backend CSV verisini filtreler ve puanlar.
4. Gemini API açıklama metni üretir.
5. Sonuçlar kullanıcıya gösterilir.
